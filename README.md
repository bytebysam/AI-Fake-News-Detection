# AI-Powered Fake News Detection

A machine learning project that classifies news articles as real or fake using Natural Language Processing (NLP) and text classification techniques.

## Project Objective

To build a machine learning pipeline for fake news detection using text preprocessing, feature extraction, model training, and evaluation.

## Models

- K-Nearest Neighbors (KNN)
- Logistic Regression
- Random Forest
- Neural Network

Project under development.
